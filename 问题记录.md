## 1. Server disconnects

```
Error: Connection lost: The server closed the connection.
...
fatal:true,
code:'PROTOCOL_CONNECTION_LOST'
```

原因：一段时间没有访问 mysql，mysql 会自动断开连接。

解决：使用连接池

```
const conn = mysql.createPool(dbConfig);
```

## 2. 跨域（CORS error）

app.js 中引入 cors 模块

```
npm i cors --save
```

```
var cors = require('cors')
app.use(cors());
```
